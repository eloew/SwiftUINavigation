# SwiftUI - Login/Logout Navigation

###### Show a way to handle navigation with Login/Logout


* Versions.
    * XCode 11.0
    * macOS 10.14.6


* Usage
    * Run the app. 
    * Enter Username (login button will appear)
    * Click login button
    * Main Screen will be shown
    * Click the UserAccount icon
    * Click Logout
    * Rinse, Repeat
